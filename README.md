# obs-stopwatch
🕒 OBS Stopwatch Pro+  A simple, professional stopwatch script for OBS Studio using Lua.  ⏱️ Count-up timer (HH:MM:SS) ⏵ Start, ⏸ Pause, ▶ Resume, 🔁 Reset 🎮 Configurable hotkeys 🖥️ Works with any text source  Lightweight, accurate, and easy to set up—ideal for streams, speedruns, and timed content.
